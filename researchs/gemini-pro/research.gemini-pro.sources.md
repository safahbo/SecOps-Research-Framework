# Sources for gemini-pro
Kaynakça ve İleri Okuma

Aşağıda SecOps araştırmasında kullanılan temel kaynaklar ve daha detaylı bilgi edinebileceğiniz bağlantılar yer almaktadır.

NIST Cybersecurity Framework

Açıklama: Siber güvenlik risklerini yönetmek için standartlar, yönergeler ve en iyi uygulamalar.

Link: https://www.nist.gov/cyberframework

Wazuh Documentation

Açıklama: Açık kaynaklı SIEM ve XDR platformu Wazuh'un resmi dokümantasyonu ve kurulum rehberleri.

Link: https://documentation.wazuh.com

Security Onion Solutions

Açıklama: Ağ güvenliği izleme (NSM) ve log yönetimi için Linux dağıtımı.

Link: https://securityonionsolutions.com

Google Cloud SecOps (Eski adıyla Chronicle)

Açıklama: Bulut tabanlı modern güvenlik operasyonları ve tehdit avcılığı platformu.

Link: https://www.google.com/search?q=https://cloud.google.com/security/operations

MITRE ATT&CK Framework

Açıklama: Siber saldırganların taktik ve tekniklerini sınıflandıran küresel bilgi tabanı.

Link: https://attack.mitre.org

OWASP (Open Web Application Security Project)

Açıklama: Yazılım güvenliği konusunda standartlar ve "Top 10" zafiyet listeleri.

Link: https://owasp.org

SANS Institute - Security Operations Center (SOC) Survey

Açıklama: SOC ekiplerinin karşılaştığı zorluklar ve istatistikler üzerine araştırmalar.

Link: https://www.google.com/search?q=https://www.sans.org/white-papers/browse/focus-areas/security-operations-center/
